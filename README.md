# VERTA — женский биохакинг-трекер

Персональное приложение для самоотслеживания с принципом **«инфрадианный цикл — главные часы»**. Заменяет зоопарк приложений (FLO, Migrebot, FatSecret, Google-таблицы, Huawei Health, RingConn, блокноты, ЕМИАС, World Class) одной системой.

Мультипользовательское веб-приложение (PWA, ставится на телефон как приложение). Деплой на Vercel, база данных — Supabase.

---

## Главная идея

Большинство трекеров работают на 24-часовом циркадном ритме — это «мужская» модель по умолчанию. У женщин поверх него лежит **инфрадианный ~28-дневный цикл**. Здесь **менструальный цикл — это главные часы**, вокруг которых организованы вес, питание, тренировки, мигрени, сон, настроение и медицина.

## Дизайн-система (VERTA)

**Фазовые цвета** меняются автоматически по фазе цикла:

| Класс на `<body>` | Фаза | Цвет акцента |
|---|---|---|
| `phase-menstrual` | Менструация (1–5 д.) | Тёплый красный |
| `phase-follicular` | Фолликулярная (6–13 д.) | Зелёный |
| `phase-ovulatory` | Овуляция (14–16 д.) | Жёлто-оранжевый |
| `phase-luteal` | Лютеиновая (17–28 д.) | Фиолетовый |

CSS-переменные: `--phase`, `--phase-soft`, `--on-phase`, `--phase-deep`, `--ink`, `--ink-2`, `--ink-3`, `--ink-4`, `--surface`, `--line`, `--warn`.

Типографика: `font-serif` (заголовки), `font-mono` (метки/цифры), `font-sans` (текст). Скруглённые карточки: `rounded-card` = `rounded-[10px]`.

Маскот — геометрическая рысь (SVG, рисуется по рёбрам с анимацией `stroke-dashoffset`, пульсирующие глаза, плавание).

## База данных (Supabase)

### Таблицы

| Таблица | Что хранит |
|---|---|
| `app_user` | Пользователи (email, invite_code, онбординг) |
| `daily_log` | Чек-ин на дату: настроение, энергия, вес, ккал, симптомы, мигрень, препараты, привычки, wearable-метрики, заметка |
| `weight_entry` | Плановый и фактический вес по датам |
| `cycle_start` | Даты начала цикла |
| `habit` | Справочник привычек с `started_month` / `ended_month` (YYYY-MM) |
| `medication` | Справочник препаратов (+ `is_as_needed` для купирующих) |
| `migraine_event` | Отдельный лог мигреней (аура, триптаны, интенсивность) — источник MigreBot |
| `import_log` | История импортов из MigreBot (файл, диапазон дат, кол-во строк) |
| `workout_log` | Лог тренировок |
| `workout_exercise` | Упражнения внутри тренировки |
| `workout_template` | Шаблоны тренировок с упражнениями |
| `weekly_schedule` | Недельное расписание тренировок |
| `calendar_event` | События календаря (тренировка/напоминание/прочее) |
| `sport_type` | Виды спорта по пользователю (с цветом) |
| `sleep_log` | Сон (RingConn): фазы, качество, длительность |
| `medical_record` | Медицинские документы (категория, врач, клиника, ссылка на Storage) |
| `balance_wheel` | Оценки колеса баланса |

Все таблицы с персональными данными изолированы по `app_user_id`. RLS включён, но фактическая защита — на уровне приложения через хелпер `byUser(query, uid)` (сервер использует `supabaseAdmin()` / service role, который обходит RLS).

### Миграции (запускаются вручную в Supabase SQL Editor)

31 миграция в `supabase/migrations/`, от `001_workout_log.sql` до `031_sleep_log_user_isolation.sql`. Ключевые вехи:

- `008_multiuser.sql`, `021_user_isolation.sql` — базовая мультипользовательская изоляция
- `022`–`026`, `029`, `031` — постепенное добавление `app_user_id` в оставшиеся таблицы
- `027_migrebot_full_csv.sql` — полный импорт истории мигреней (438 записей, 2021–2026)
- `030_import_log.sql` — история импортов из MigreBot

## Запуск локально

```bash
cd web
npm install
# создать .env.local с переменными:
#   NEXT_PUBLIC_SUPABASE_URL
#   NEXT_PUBLIC_SUPABASE_ANON_KEY
#   SUPABASE_SERVICE_ROLE_KEY
#   PETAPP_INVITE_CODE
npm run dev                  # http://localhost:3000
```

Без Supabase — приложение падает на auth. Нужна живая база.

## Статус

🟢 **В продакшене.** Все основные экраны реализованы, задеплоены на Vercel с реальными данными, мультипользовательская изоляция данных проведена по всем таблицам.

- Медкнижка — каталог документов с категориями, приватное хранилище
- Импорт из Migrebot — CSV → история мигреней, с логом импортов
- Инсайты по мигреням, носимые устройства (RingConn)
- Колесо баланса
- PWA (ставится на телефон)
